BossProgress 1.0.7

Plugin for bbDKP 1.2.8 or higher. it gives a Bossprogress page and a Portal block with a popup. Zones and bosses can be added or deleted through the ACP. Integrates with the Event ACP, so you can add new events by picking a Bossprogress Zone. Zone and Boss images in the Images folder (21 mb in total); Excel sheet included if you want to customise the installer and integrate large amounts of boss/zone info into the database.

Feature list

  • Raid progress page
  • Portal Block with Raid status and Boss tooltips
  • ACP Zone editor
  • ACP Boss editor
  • Event Creation easier : after you install Bossprogress, the Event creation ACP will have a new zone import pulldown.
  • Boss images included (color and b/w)
  • Zone images included (color, sepia and grey)

Installation

Quick install guide

  1. Copy the whole Bossprogress folder to /store/mods
  2. log in to your ACP, go to the Automod tab
  3. Click Install to let Automod copy the files and do the core edits.
    4. Log into your FTP and copy the /images/Bossprogress folder manually to your /Images folder.
  4. Browse to install/index.php and run the Bossprogress plugin table installer/updater.
  5. Delete the root/install folder
